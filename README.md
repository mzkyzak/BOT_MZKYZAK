## 🌟 Daftar Fitur (Features List)

Bot ini dilengkapi dengan ratusan perintah yang dibagi ke dalam beberapa kategori menu lengkap, antara lain:

- 👑 **Owner & Panel Menu** : Manajemen bot, pengaturan VPS/Linode, panel Pterodactyl, broadcast, dan konfigurasi database.
- 👥 **Group Management** : Sistem antilink, kick, promote/demote, hidetag, dan pengaturan grup otomatis.
- 🤖 **AI & Gemini Integration** : Chat AI (OpenAI, Gemini), manipulasi gambar (toanime, removal, faceblur), dan OCR.
- 📥 **Downloader Menu** : Unduh media dari TikTok, Instagram, Facebook, YouTube (MP3/MP4), Google Drive, dan Mediafire.
- 🛠️ **Tools & Converter** : Ubah format media (toimg, toaudio), efek suara (bass, nightcore, dll), dan text-to-speech.
- 🎮 **Fun & Games** : Game tebak-tebakan (tebak gambar, asah otak, family100), cek khodam, dan hiburan lucu lainnya.
- 💌 **Menfess & Anonymous** : Fitur kirim pesan rahasia (confess/menfess) antar pengguna.
- 📰 **News & Search** : Pencarian Google, Pinterest, Playstore, dan portal berita terkini (CNN, Detik, Kompas, dll).
- 🎨 **Maker & Random** : Pembuat stiker, Brat generator, gambar random (Anime, Cosplay, Wallpaper), dan Random Video.
- 📖 **Islam & Primbon** : Jadwal sholat, Al-Qur'an, ramalan jodoh, zodiak, dan fitur Islami lainnya.
- 🛒 **Store & Push Kontak** : Fitur khusus untuk jualan (addlist, dellist, payment), dan push kontak otomatis.
- ⚔️ **Genshin Impact** : Informasi build karakter, material, map, dan data lengkap Genshin Impact.
- 🎭 **Quotes & Sad Menu** : Kumpulan kata-kata motivasi, pantun, puisi, dan ratusan audio *sad vibes*.

> 💡 *Ketik `.menu` di dalam chat WhatsApp untuk melihat daftar lengkap semua command yang tersedia.*

## 📦 Cara Install & Menjalankan Bot

### 1. Persiapan
Pastikan kamu sudah menginstal **[Node.js](https://nodejs.org/en/download/)** (Disarankan versi 16 ke atas) di komputer atau server VPS kamu.

### 2. Instalasi
Buka Terminal/CMD, lalu jalankan perintah berikut secara berurutan:

```bash
# 1. Kloning repository ini
git clone [https://github.com/mzkyzak/BOT_MZKYZAK.git](https://github.com/mzkyzak/BOT_MZKYZAK.git)

# 2. Masuk ke direktori bot
cd BOT_MZKYZAK

# 3. Install semua module (dependencies)
npm install

# 4. Jalankan bot
npm start
```
