🤖 MZKYZAK BOT

Bot WhatsApp sederhana berbasis Baileys yang dibuat menggunakan JavaScript.

🚀 Fitur

- Login menggunakan Pairing Code (tanpa QR)
- Sistem menu & command
- Support plugin
- Ringan dan cepat

📦 Cara Install

pkg update && pkg upgrade
pkg install nodejs git
git clone https://github.com/username/mzkyzak-bot
cd mzkyzak-bot
npm install
node index.js

📱 Cara Penggunaan

1. Jalankan bot
2. Masukkan nomor WhatsApp (contoh: 62xxx)
3. Masukkan kode pairing
4. Bot siap digunakan

⚠️ Catatan

- Pastikan koneksi internet stabil
- Jika terjadi error, hapus session lalu login ulang:

rm -rf session

🛠 Teknologi

- Node.js
- Baileys
- Pino Logger
