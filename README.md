# 🤖 MZKYZAK BOT

Bot WhatsApp sederhana, ringan, dan cepat yang dibangun menggunakan library [Baileys](https://github.com/WhiskeySockets/Baileys) dan Node.js.

![NodeJS](https://img.shields.io/badge/Node.js-v16%2B-green)
![Baileys](https://img.shields.io/badge/Baileys-Library-blue)

## 🚀 Fitur Utama

- 🔑 **Login Pairing Code** (Aman tanpa perlu scan QR Code)
- 📋 **Sistem Menu & Command** yang rapi
- 🧩 **Support Plugin** 
- ⚡ **Ringan & Cepat** 

## ⚠️ Catatan Penting
> Saat ini bagian fitur **Download (YTMP3/4), Quotes, dan AI** masih dalam tahap pengembangan dan perbaikan (*Bugs fixing*).
- Pastikan koneksi internet stabil.
- Jika terjadi error saat login, hapus folder session lalu login ulang.

## 📦 Cara Install & Menjalankan Bot

### 1. Persiapan
Pastikan kamu sudah menginstal **[Node.js](https://nodejs.org/en/download/)** di komputer/server kamu.

### 2. Instalasi
Buka Terminal/CMD, lalu jalankan perintah berikut secara berurutan:

```bash
# Kloning repository ini
git clone [https://github.com/mzkyzak/BOT_MZKYZAK.git](https://github.com/mzkyzak/BOT_MZKYZAK.git)

# Masuk ke direktori bot
cd BOT_MZKYZAK

# Install semua module (dependencies)
npm install

# lalu jalankan bot
npm start
